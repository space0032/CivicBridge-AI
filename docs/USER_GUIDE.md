# User Guide

Welcome to CivicBridge AI! This guide will help you get started with using the application.

## Getting Started

### Creating an Account

To get started, you'll need to create an account. You can do this by clicking the "Register" button on the home page. You'll be asked to provide your name, email address, and a password.

### Logging In

Once you've created an account, you can log in by clicking the "Login" button on the home page. You'll be asked to provide your email address and password.

## Features

### Programs

The "Programs" page displays a list of government programs that you can apply for. You can filter the programs by category and region. To view the details of a program, click on the program card.

### Healthcare

The "Healthcare" page displays a list of healthcare facilities. You can filter the facilities by type and whether they offer free services. To view the details of a facility, click on the facility card.

### Voice Search

The "Voice Search" page allows you to search for programs and healthcare facilities using your voice. To use this feature, click the microphone button and speak your query.

### Apply Now

The "Apply Now" feature allows you to apply for a program directly from the application. To use this feature, click the "Apply Now" button on the program details page. You'll be taken to a page where you can fill out the application form and submit it.

### My Applications

The "My Applications" page displays a list of the applications that you've submitted. You can view the status of your applications on this page.
